Um das Spiel zu spielen:
Im Spiel Ordner auf Platform runner.exe clicken und dann auf play.

Regeln (siehe abstract im PDF):
Das Ziel des Spiels ist es, länger als 3 Bots auf 1-3 Platformebene/n zu überleben. 
Eine Platformebene besteht aus kleinen Platformen, welche zerstört werden, wen ein Bot oder der Spieler über sie gelaufen ist.
Wenn man am Rand ankommt wird man zurück zum Spielfeld "geschüpft".

Steuerung:
Maus bewegen: Spieler drehen
Leertaste: Springen
Escape: Pause
