Um das Spiel zu spielen:
Im Spiel Ordner auf Platform runner.exe clicken und dann auf play.

Regeln:
Überlebe länger als 3 Bots auf 1-3 Platformebene/n. 
Eine Platformebene besteht aus kleinen Platformen, welche zerstört werden, nachdem ein Bot oder der Spieler über sie gelaufen ist.

Steuerung:
Maus bewegen: Spieler drehen
Leertaste: Springen
Escape: Pause
